# Idea Opportunity Radar
# 想法機會雷達

## 1. Purpose｜目的

這份文件用來建立 AVIN 的第一版 Idea Opportunity Radar。

它不是銷售清單。

它不是產品待辦清單。

它也不是要立刻判斷哪些想法可以變現。

這份文件的目的，是把 AVIN 過去與現在反覆出現的想法放到同一張雷達圖上，觀察：

- 哪些只是靈感
- 哪些開始形成方向
- 哪些已經有 workflow 證據
- 哪些已經有公開輸出
- 哪些開始出現外部訊號
- 哪些具備可複用資產潛力
- 哪些未來可能值得進一步做商機驗證

核心原則：

> 不是急著變現。  
> 是先建立商機雷達。

---

## 2. Relationship to Self-Ecosystem Analysis｜與自我生態分析的關係

AI Digital Footprint OS 的原始動機，是分析 AVIN 自己的生態、想法、行動軌跡與外部回饋，並用這些訊號驗證方向。

Idea Opportunity Radar 是這個原始動機下的第一個分析工具。

它幫助 AVIN 把日常累積的想法從腦中拉出來，放到可觀察的位置。

它要回答的不是：

> 這個可以賣嗎？

而是：

> 這個想法是否值得繼續觀察、實驗、公開或驗證？

---

## 3. Relationship to Productization Layer｜與產品化判斷層的關係

Productization Layer 是後段判斷層，用來判斷完成的 workflow 是否具備可複用資產潛力。

Idea Opportunity Radar 則更前面。

它處理的是：

- 還沒形成 workflow 的想法
- 剛開始反覆出現的方向
- 已經有局部實驗但還沒有完整驗證的題目
- 有可能往 workflow / case study / SOP / template / mini kit 發展的原料

兩者關係是：

Idea Opportunity Radar  
→ Workflow Experiment  
→ Public Output  
→ Feedback Signal  
→ Productization Quick Scorecard  
→ Productization Judgment

中文理解：

想法機會雷達  
→ 工作流實驗  
→ 公開輸出  
→ 回饋訊號  
→ 產品化快速評分  
→ 產品化判斷

所以 Idea Opportunity Radar 不是 Productization Queue。

它是更早期的觀察雷達。

---

## 4. What This Radar Is Not｜這份雷達不是什麼

這份雷達不是：

- 銷售清單
- 產品 backlog
- Agent Product Queue
- 變現計畫表
- Notion 自動化需求表
- 社群選題表
- 爆款公式表
- 立即執行清單

它也不應該把每個想法都推向產品化。

它只負責觀察：

> 哪些想法開始反覆出現？  
> 哪些想法和 AVIN 的能力與定位有關？  
> 哪些想法有可能形成可驗證的機會？

## 5. Evaluation Fields｜評估欄位

### Idea｜想法

---

是否有人真的遇到？

### Current Evidence｜目前證據

目前有哪些證據支持這個想法值得觀察？

可能包含：

- 已做過 workflow
- 已寫成文件
- 已公開發布
- 已收到數據
- 已有案例
- 自己反覆使用
- 已經能清楚說明

### External Signal｜外部訊號

- 有人私訊問模板

- 不同平台重複出現同類問題

- 按讚
- 泛泛稱讚
- 單次好奇
- 沒有脈絡的收藏
- 單篇高曝光

商業出口可能包含：

- 付費模板
- 迷你工作流包
- 顧問切入方案
- 教學內容
- 工作坊
- 垂直產業 workflow kit

### Next Validation Step｜下一步驗證

這個想法下一步最小驗證是什麼？

- 建立一份 GitHub 文件
- 做一個 workflow demo
- 問 3 個目標受眾
- 收集私訊或留言問題
- 做一份 checklist
- 先當 case study 文件化

---

### Status｜狀態

建議狀態：

- Raw Idea｜原始想法
- Pattern Emerging｜模式浮現
- Workflow Tested｜已測 workflow
- Signal Detected｜已偵測訊號
- Commercial Candidate｜商機候選
- Archived｜封存

---

## 6. Status Definitions｜狀態定義

### Raw Idea｜原始想法

只是靈感或初步念頭。

還沒有明確問題、受眾或實驗。

---

### Pattern Emerging｜模式浮現

這個想法反覆出現，開始和 AVIN 的能力、定位或外部趨勢產生關聯。

但還沒有完整 workflow 或明確外部訊號。

---

### Workflow Tested｜已測 workflow

這個想法已經被做成某種 workflow、文件、內容或實驗。

但外部訊號還不足。

---

### Signal Detected｜已偵測訊號

外部開始出現有效訊號。

例如有人詢問、私訊、描述相同問題、要求模板、願意測試。

---

### Commercial Candidate｜商機候選

這個想法已經具備：

- 明確問題
- 具體受眾
- AVIN 的不公平優勢
- 可複用資產形式
- 外部有效訊號
- 可承受的支援風險

這時候才可以進入更正式的商機驗證。

---

### Archived｜封存

這個想法目前不值得繼續追。

可能原因：

- 不符合定位
- 沒有受眾
- 問題不夠明確
- AVIN 沒有優勢
- 支援成本太高
- 外部訊號不足

---

### 7.1 AI Digital Footprint OS

- 學了很多，但沒有留下證據
- 測了很多工具，但沒有形成方法
- 發了很多內容，但沒有回收判斷
- 想轉職或建立定位，但沒有可展示的系統

#### Possible Audience｜可能受眾

- AI 學習者
- 個人創作者
- 自由工作者
- 想建立公開履歷的人
- 想把 AI workflow 文件化的人
- 想從內容輸出累積專業資產的人

#### My Unfair Advantage｜我的不公平優勢

AVIN 正在實際建立這套系統，且已串接 Notion、GitHub、社群發布、API 數據回收與個人網站展示。

AVIN 也有內容策略、短影音、社群與數據優化背景，能把這套系統轉成一般人看得懂的敘事。

#### Current Evidence｜目前證據

- 已建立 AI Digital Footprint OS 主幹
- 已完成多份 OS 文件
- 已有 GitHub repo
- 已有個人網站展示
- 已完成 Threads API 自動成效回收案例
- 已建立 Productization Layer
- 已建立 Self-Ecosystem Analysis 原始動機文件

#### External Signal｜外部訊號

目前仍偏早期。

需要觀察是否有人詢問：

- 這套系統怎麼搭
- GitHub / Notion 怎麼串
- AI 學習紀錄怎麼變公開資產
- 個人 AI portfolio 怎麼建立

#### Reusable Asset Potential｜可複用資產潛力

高。

可能變成：

- OS Case Study
- Public AI Learning Log Template
- AI Workflow Portfolio SOP
- Notion + GitHub Documentation Kit
- Personal Digital Footprint Kit

#### Commercial Potential｜商機潛力

中高，但需要外部訊號。

未來可能成為個人 AI 學習者、創作者或轉職者的公開資產系統方法論。

> 哪些 AI 工具值得被放進實際 workflow 測試？

許多人每天看到大量 AI 工具與新聞，但不知道：

- 哪些值得看
- 哪些只是噪音
- 哪些適合自己 workflow
- 哪些可以形成內容
- 哪些可以變成長期觀察題目

#### Possible Audience｜可能受眾

- AI 學習者
- 創作者
- 行銷人
- 獨立工作者
- 想導入 AI workflow 的小型團隊

#### My Unfair Advantage｜我的不公平優勢

AVIN 也有內容策略能力，能把工具觀察轉成 workflow 判斷，而不是工具清單。

#### Current Evidence｜目前證據
