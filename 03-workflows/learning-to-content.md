# Learning To Content

## Goal

Turn AI learning into public-facing content without losing substance.

## Workflow

1. Capture the learning session.
2. Write a structured learning note.
3. Identify one idea worth sharing publicly.
4. Use the AI research breakdown skill if the topic is still fuzzy.
5. Use the multi-platform content skill to create channel-specific drafts.
6. Run a credibility review before publishing.
7. Publish the strongest version and link back to the source asset.

## Inputs

- learning note
- selected takeaway
- target channels

## Outputs

- one long-form reference asset
- one or more short-form content drafts

## Review Questions

- Did the learning produce an asset
- Did the asset produce content
- Does the content point back to something durable
