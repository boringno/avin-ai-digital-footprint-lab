# Notion To GitHub

## Goal

Convert a raw note or experiment from Notion into a durable GitHub document.

## Workflow

1. Start with a note, experiment log, or idea capture in Notion.
2. Distill it into a reusable topic, workflow, or case study.
3. Choose the correct destination in the repo:
   learning note, workflow, skill, case study, or template.
4. Rewrite the content in Markdown with explicit headings.
5. Add links, examples, and practical takeaways.
6. Commit the document so the insight becomes versioned and referenceable.

## Inputs

- raw note
- target folder in repo
- desired artifact type

## Output

- one durable GitHub document

## Review Questions

- Will this still make sense in three months
- Is the content structured enough to be cited or reused
- Did the note become a real asset rather than a storage dump
