# LinkedIn Website Entry Update

日期：2026-05-11  

分類：平台更新 / LinkedIn  

狀態：LinkedIn 已加入 GitHub Pages 個人網站入口

---

## 一、更新內容

目前 LinkedIn 已新增網站入口，對外可直接導向 AVIN 的 GitHub Pages 個人網站首頁。

- Website URL：
  `https://boringno.github.io/avin-ai-digital-footprint-lab/`

---

## 二、平台關係

這次更新之後，三個公開入口之間的關係更清楚：

- GitHub repo：
  作為公開證據層，承接文件、commit、workflow 與結構化內容
- GitHub Pages：
  作為對外可瀏覽的首頁入口，承接個人網站展示層
- `llms.txt`：
  作為 AI / LLM 可讀的導讀入口，補上機器可讀層
- LinkedIn：
  作為曝光層，現在已直接接上網站入口層

---

## 三、階段意義

這個節點代表 LinkedIn 不再只是單獨的個人資料頁，而是已經能把外部流量導向 AVIN AI Digital Footprint OS 的公開入口。

也就是說，LinkedIn 曝光層、GitHub 證據層、GitHub Pages 入口層與 `llms.txt` 機器可讀層，現在已經開始形成可互相支援的公開數位足跡結構。
