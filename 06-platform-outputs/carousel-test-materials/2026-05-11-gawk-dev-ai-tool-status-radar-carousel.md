# 你的 AI 工具今天正常嗎？

日期：2026-05-11  

分類：Carousel Test Material / External Skill Test  

狀態：測試素材

---

## 1. Title

你的 AI 工具今天正常嗎？

---

## 2. Context

- 這是 External Carousel Skill Test Plan 的第一個測試素材
- 主題來自 Gawk Dev / AI coding stack status radar
- 用來測試 `social-media-carousel` 等外部 Skill 是否能產出更好的 IG 輪播結構與視覺 brief

---

## 3. Core Insight

- 當 AI 變成工作流的一部分，它就不只是工具，而是基礎設施
- 基礎設施需要被監控
- 使用者需要的不只是 AI 工具清單，也需要 AI 工具狀態雷達

---

## 4. Target Audience

- 使用 Claude、OpenAI、Cursor、GitHub Copilot、Windsurf 等 AI 工具工作的人
- AI coding / AI workflow 使用者
- 會遇到 AI 工具變慢、卡住、不穩、回應異常的人
