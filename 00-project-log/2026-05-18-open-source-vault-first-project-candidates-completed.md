## 2. 為什麼從 candidate batch 轉成 project-template

Candidate batch 適合做第一層分流，但它還是批次視角。

轉成 `project-template` 的價值是：

- 每個候選有單獨紀錄
- 可以補上權限、風險、use case、decision 與 lifecycle
- 可以更清楚地進入後續安全檢查與試用流程

這樣候選就不只是表格裡的一列，而是可追蹤、可審核、可延後決策的單一項目。
