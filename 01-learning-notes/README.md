# Learning Notes

This folder stores structured notes from AI learning sessions.

## Purpose

The goal is not to save everything.

The goal is to save what becomes reusable:

- important concepts
- useful comparisons
- tested workflows
- implementation lessons
- risks, limits, and tradeoffs

## Suggested Note Format

Use the template in [learning-log-template.md](../05-templates/learning-log-template.md).

Each note should aim to answer:

- what did I learn
- why does it matter
- where can this be applied
- what should be turned into a workflow, prompt, or public post

## Naming Suggestion

Use a date and topic:

```text
2026-05-08-openai-agents-sdk-notes.md
2026-05-08-rag-evaluation-basics.md
```

## Quality Bar

A good learning note should be understandable later without relying on the original chat, tab, or video.
